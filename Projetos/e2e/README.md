# Testes e2e

Teste e2e desenvolvido para aplicação do Projeto ...... 


## Features a serem testadas 

- Login
- AppBar
- DarkTheme
- Listagem dos Processos
- CRUD de Processo
- Busca de Processo por Assunto


## Bibliotecas utilizadas para a automação 

- Cypress 
- Keycloack
- Schema Validador


## Executar os cenários

- Abrir o Cypress

```
    npx cypress open
```

- Rodar os Testes

```
    npx cypress run
```
